E-Commerce Clothing Website

Technology Used: vs code , HTML, CSS, JavaScript

  • This is an online clothing store offering various fashion items for men and women , providing a seamless shopping experience on laptop and phone.
  • The home page serves as the entry point for users , featured products , new arrivals. Responsive element include products, banner, newsletter, footer with contact and various information about the website.
  • The shop page display all available products in an organized grid layout. User can click an individual products for full contents. Users can filter products by categories, sizes and price .
  • The blog page serves as a space to post articles, fashion tips, and brand update. Features include a grid layout of blog post tittles, images, footer with contacts and various information about the website. The page is designed with a clean and easy-to read layout, adapting to mobile phone and tablets screen also
  • The about us page introduce the brand, its mission and vision. The design is clean with text and visuals that communicate with brand’s identity.
  • In contact page we give our address of store. In contact page we uses Web3forms API for secure, backend free form submissions. Features include banner , map , footer with contact and other information about the website.
  • The cart page allows user to view, remove or update items in their cart. Shows a summary of selected items, quantities, and total price. Future implementation of payment gateway.

Diwali Sales Data Analysis

Technology Used: Python, Kaggle,libraries(pandas, NumPy, Seaborn, Matplotlib), Jupyter notebook

  • This project is a comprehensive Exploratory Data Analysis (EDA) of a retail dataset focusing on Diwali sales, using Python and key libraries such as Pandas, NumPy, Seaborn, and Matplotlib.
  • The goal is to extract business insights related to customer behavior, product demand, and regional sales performance.
  • Data Import and Cleaning: Loaded the dataset from a CSV file using Pandas. Removed unnecessary columns (Status, unnamed1) and null values in the Amount column. Changed the data type of the Amount column to integer for accurate analysis.
  • Visualizations Used: Count plots for Gender, Age Group, Marital Status, Occupation, and Product Categories. Bar plots for state-wise orders and revenue. Horizontal and vertical bar charts for top product sales.
